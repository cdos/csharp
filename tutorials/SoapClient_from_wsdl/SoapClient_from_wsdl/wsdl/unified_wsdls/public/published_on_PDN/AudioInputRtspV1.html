AudioInputRtsp - Web Service


$Id: AudioInputRtspV1.wsdl 474 2010-12-22 19:25:11Z $
This service allows a client to query available and outgoing audio streams and to request connections to those streams.
Target Namespace:
urn:schemas-pelco-com:service:AudioInputRtsp:1

    1. Connect

      Initiates a connection to a specific audio channel.
      SOAP action:
      urn:schemas-pelco-com:service:AudioInputRtsp:1#Connect
      Input:
      [Connect]
      This is a request to the Connect action.
      • sourceURI [anyURI]
        The URI of the original source of the audio data. e.g. uuid:EnduraEncoder-CU6350-5/urn:pelco-com:serviceId:AudioInputRtsp-1 e.g. uuid:ca522d4a-d26f-446a-8e7e-ccdb6127fd2d/urn:pelco-com:serviceId:AudioInputRtsp-16 For audio connections: e.g. rtsp://10.221.224.120/audioStreamIdentification
      • channelId [int]
        A channel's unique identifier.
      • sessionTimeoutSeconds [int]
      Output:
      [ConnectResponse]
      This is a response to a Connect request.
      • streamId [int]
        A stream session's unique identifier.
      • streamStatus [string]
        A stream's current status. Valid values are the following:
        • Ok
        • Bad ChannelId: the is associated with an invalid channel identifier.
        • Invalid Url: the stream has an invalid URL.
        • Unsupported Format: the stream's format is currently unsupported.
        • Failed: the stream failed.
