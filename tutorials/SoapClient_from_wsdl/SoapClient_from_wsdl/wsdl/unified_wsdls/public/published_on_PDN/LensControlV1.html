This service allows control of a camera's lens. Specifically the following elements are configurable:

  • configuration of the iris
  • configuration of lens focus
  • magnification level of the lens
  • maximum angle of view of the lens

Target Namespace:
urn:schemas-pelco-com:service:LensControl:1
[toc]

Operations

    1. AutoFocus

      Turns auto focus on/off.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#AutoFocus
      Input:
      [AutoFocus]
      Represents a request to the AutoFocus action.
      • onOff [int]

        State of the auto focus setting of the lens. Controls which aspects of a picture are sharp. Must be one of:

        • 0 for OFF
        • 1 for ON

      Output:
      [AutoFocusResponse]
      Represents a response from the AutoFocus action.

    2. AutoIris

      Turns auto iris on/off.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#AutoIris
      Input:
      [AutoIris]
      Represents a request to the AutoIris action.
      • onOff [int]

        State of the auto iris setting of the lens. The iris controls exposure or how bright the video frames will appear. Must be one of:

        • 0 for OFF
        • 1 for ON

      Output:
      [AutoIrisResponse]
      Represents a response from the AutoIris action.

    3. Focus

      Focuses the lens near/far, continuously until a limit is reached.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#Focus
      Input:
      [Focus]
      Represents a request to the Focus action.
      • nearFar [unsignedInt]

        This action sets the state of the focus setting of the lens, which moves the focus motor in a particular direction. The motor keeps moving until it hits a limit or stop is called. Must be one of:

        • 0 for STOP
        • 1 for NEAR
        • 2 for FAR

      Output:
      [FocusResponse]
      Represents a response from the Focus action.

    4. GetMag

      Gets the current magnification of the lens.

      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#GetMag
      Input:
      [GetMag]
      Represents a request to the GetMag action.
      Output:
      [GetMagResponse]
      Represents a response from the GetMag action.
      • magnification [unsignedInt]
        State of the focus setting of the lens. The higher the value, the higher the magnification.

    5. GetMaxAOV

      Returns the maximum possible angle-of-view of the lens.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#GetMaxAOV
      Input:
      [GetMaxAOV]
      Represents a request to the GetMaxAOV action.
      Output:
      [GetMaxAOVResponse]
      Represents a response from the GetMaxAOV action.
      • aovMax [unsignedInt]
        Maximum possible angle-of-view of the lens. Determines the maximum possible area that the camera can capture. Units are in centidegrees, so an aov of 90 degrees would be 9000.

    6. GetMaxDigitalMag

      Returns the maximum digital magnification of the lens.

      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#GetMaxDigitalMag
      Input:
      [GetMaxDigitalMag]
      Represents a request to the GetMaxDigitalMag action.
      Output:
      [GetMaxDigitalMagResponse]
      Represents a response from the GetMaxDigitalMag action
      • magnification [unsignedInt]
        The maximum digital magnification of the lens. This determines the ability to see near and far.

    7. GetMaxOpticalMag

      Returns the maximum optical magnification of the lens.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#GetMaxOpticalMag
      Input:
      [GetMaxOpticalMag]
      Represents a request to the GetMaxOpticalMag action.
      Output:
      [GetMaxOpticalMagResponse]
      Respresents a response from the GetMaxOpticalMag action.
      • magnification [unsignedInt]
        The maximum optical magnification of the lens. This determines the ability to see near and far.

    8. Iris

      Opens/closes the lens iris, continuously until a limit is reached.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#Iris
      Input:
      [Iris]
      Represents a request to the Iris action.
      • openClose [unsignedInt]

        Sets the state of the iris of the lens, which starts moving the iris in a particular direction. Movement continues until a limit is reached or STOP is called. Must be one of:

        • 0 for STOP
        • 1 for CLOSE
        • 2 for OPEN

      Output:
      [IrisResponse]
      Represents a response from the Iris action.

    9. SetMag

      Sets the current magnification of the lens. Magnification must be between 1 and MaxMag value.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#SetMag
      Input:
      [SetMag]
      Represents a request to the SetMag action.
      • magnification [unsignedInt]
        The maximum optical magnification of the lens. This determines the ability to see near and far.
      Output:
      [SetMagResponse]
      Represents a response from the SetMag action.

    10. Stop

      Stops all currently executing lens operations.

      Anytime you explicitly call the Iris, Focus, or Zoom actions; the camera will continue executing them (opening the iris, zooming in, ...) regardless of whether the camera has reached the maximum or minimum state (maximum zoom or focus value). At some point, this action must be called.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#Stop
      Input:
      [Stop]
      Represents a request to the Stop action.
      Output:
      [StopResponse]
      Represents a response from the Stop action.

    11. Zoom

      Changes the magnification of the lens, continuously zooming in or out until a limit is reached.
      SOAP action:
      urn:schemas-pelco-com:service:LensControl:1#Zoom
      Input:
      [Zoom]
      Represents a request to the Zoom action.
      • inOut [unsignedInt]

        Controls the lens zoom in and out. Must be one of:

        • 0 for STOP
        • 1 for OUT
        • 2 for IN

      Output:
      [ZoomResponse]
      Represents a response from the Zoom action.
